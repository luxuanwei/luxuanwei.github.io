Docker常用命令

发表于 2016-03-15

Docker Linux安装篇

sudo apt-get uodate
sudo apt-repostry ppa:docker-maint/testing
sudo apt-get install docker.io

Docker 基本概念

Docker Daemon 运行在一个主机上
Docker Client 访问在Docker的渠道
Docker Images 一个只读模板
Docker Registry 存放Images的仓库
Docker Containers 文件夹/实例

Docker 常用命令

docker version      查看容器版本
docker build -t     根据dockerfile创建镜像
docker images       查看本机镜像
docker history      查看镜像创建各种版本
docker run -d       创建镜像的容器
docker ps           查看正在运行的容器
docker pull         拉取镜像
docker push         上传镜像
docker rmi $(docker images -q)   删除本机镜像 
docker stop $(docker ps -a -q)   停止本机容器
