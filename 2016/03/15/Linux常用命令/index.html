ubuntu常用命令

apt-get update  更新系统
apt-get upgrade 升级已安装所有软件包
apt-get install -y nano 安装nano编辑器
cd ..           进入上一层
wget [url]      下载地址
tar xzf         解压
./configure     源代码安装
make [option: target_name]  编译安装
touch           新建文件
mkdir           新建文件夹
vi/nano         编辑文件
